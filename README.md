# Root Cause Analysis - Uber Case Study

This project analyzes Uber's ride cancellation and car availability issues using data-driven approaches.

## Problem Statement

Uber has received complaints about ride cancellations by drivers and non-availability of cars for specific routes. The uneven supply-demand gap for cabs between City and Airport is affecting customer relationships and causing revenue loss.

## Objective

Identify the root cause of ride cancellations and car non-availability, and recommend solutions to improve the supply-demand balance.

## Dataset

The analysis uses `uber-data.csv` containing ride booking data with request details, pickup points, driver information, trip status, and timestamps.

## Analysis

The project examines cancellation patterns, supply-demand gaps, time-based trends, and route-specific issues to provide actionable insights for optimizing Uber's operations.
